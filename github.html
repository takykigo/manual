1. GitHubへのログイン方法
▶ GitHubにアクセス

https://github.com にアクセス。

▶ アカウント作成（初めての場合）
  • 右上の「Sign up」をクリック
  • ユーザー名、メール、パスワードを入力
▶ ログイン
  • 右上の「Sign in」からログイン
2. リポジトリの作成
  • 右上「＋」→「New repository」
  • 名前例: my-html-page
  • Public を選択（GitHub Pages用）
  • READMEにチェック → Create repository
3. HTMLファイルの追加方法
方法A: GitHub上で作成
  1. 「Add file」→「Create new file」
  2. ファイル名を index.html に
  3. HTMLを記述（例は下）
  4. 「Commit new file」をクリック
<!DOCTYPE html> <html> <head><title>私のウェブサイト</title></head> <body> <h1>こんにちは、世界！</h1> <p>これはGitHub Pagesで公開された私の最初のページです。</p> </body> </html>
方法B: ファイルをアップロード
  1. 「Add file」→「Upload files」
  2. index.html をドラッグ＆ドロップ
  3. 「Commit changes」をクリック
4. HTMLの修正
  • 「Code」→ index.html をクリック
  • 右上の鉛筆マークで編集
  • 下にスクロールし「Commit changes」で保存
5. GitHub Pagesで公開
  • 「Settings」→「Pages」
  • Source: mainブランチ + /(root)
  • 「Save」ボタンをクリック
  • URL表示: https://ユーザー名.github.io/リポジトリ名/

変更を加えるたびに自動で再デプロイされます

6. URLを第三者に共有
  • 表示されたGitHub PagesのURLをコピー
  • メールやSNSで共有 → そのままブラウザで閲覧可能
