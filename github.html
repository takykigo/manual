① GitHubアカウントの作成とログイン

🌐 GitHubにアクセス
https://github.com にアクセスします。
🆕 アカウント作成（初めての方）
  • 右上の「Sign up」をクリック
  • ユーザー名、メール、パスワードを入力し登録
  • メール認証後にアカウントが作成されます
🔐 ログイン（既にアカウントがある場合）
右上の「Sign in」からログイン

② 新しいリポジトリの作成

📁 「+」→「New repository」
GitHub右上の「+」アイコンから新しいリポジトリを作成します。
📝 入力内容
  • Repository name: 例: my-html-site
  • Public を選択（GitHub Pagesでは公開が必須）
  • Add README: チェックを入れてOK
✅ 「Create repository」クリック

③ HTMLファイルの追加

🧾 方法A: GitHub上で直接作成
  • 「Add file」→「Create new file」
  • ファイル名：index.html
  • 下記のHTMLコードを入力
<!DOCTYPE html> <html> <head><title>My Webpage</title></head> <body> <h1>こんにちは、世界！</h1> <p>GitHub Pagesで公開中です。</p> </body> </html>
「Commit new file」で保存
📤 方法B: ファイルをアップロード
  • 「Add file」→「Upload files」
  • PCから index.html をアップロード
  • 「Commit changes」で保存

④ HTMLの修正方法

✏️ 既存ファイルを編集
  1. 「Code」タブで index.html を開く
  2. 右上の鉛筆アイコン（Edit this file）をクリック
  3. 内容を修正し、「Commit changes」で保存

⑤ GitHub Pagesで公開

🚀 Settings → Pages 設定
  • リポジトリ上部の「Settings」をクリック
  • 左メニュー「Pages」を選択
📂 ブランチとルートを設定
  • Source: 「main」ブランチ
  • フォルダ: 「/（root）」
  • 「Save」ボタンをクリック
🌍 公開URLを確認
「Your site is live at https://ユーザー名.github.io/リポジトリ名/」というURLが表示されます。

⑥ URLを共有する方法

🔗 URLをコピーして共有
表示されたURL（例：https://takylkgo.github.io/my-html-site/）をコピーし、メールやSNSで共有できます。

⚠️ 注意事項とヒント

HTMLを更新してもすぐに反映されないことがあります。
対応策： 5～10分ほど待つ / ブラウザのキャッシュをクリア（Ctrl+F5 または Cmd+Shift+R）
GitHub Pagesは静的サイト向けのホスティングサービスです。HTML, CSS, JavaScript に対応しています。
